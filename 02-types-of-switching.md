# Topic 2: Types of Switching

There are three historical methods used to transport data units across networks:

1. **Circuit Switching:** Establishes a dedicated, continuous physical path between sender and receiver before communication begins (e.g., traditional landline phones).
2. **Packet Switching:** Breaks data into smaller pieces called packets/frames. Each packet contains headers and travels independently across the network, reassembling at the destination (Modern network standard).
3. **Message Switching:** Sends the complete data block from node to node using a "Store-and-Forward" mechanism. Intermediate nodes hold the entire message before sending it ahead (Legacy standard).
